# ☕ Coffee Sales Dashboard (Excel Project)

## 📌 Project Overview  
This project is a **Sales Dashboard** built entirely in **Microsoft Excel**.  
It connects and analyzes three key tables — **Order, Customer, and Product** — to provide meaningful business insights.  

The dashboard highlights:  
- 📈 **Total Sales Over Time**  
- 🌍 **Sales by Country**  
- 👥 **Top Customers**  

Interactive filters (slicers) used:  
- 📅 Month of Order Date  
- ☕ Coffee Roast Type  
- 📏 Size  
- 🎫 Loyalty Card  

---

## 🛠 Techniques Used  
- **VLOOKUP / XLOOKUP** → To join Order, Customer, and Product tables  
- **Slicers** → For dynamic filtering  
- **PivotTables & Charts** → For clean visualization  
- **Diagram Mapping** → To showcase table relationships
